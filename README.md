基于 Claude Code 与 MiMo 系列模型构建的个人开发全流程智能助手，实现从需求分析到部署上线的端到端自动化。

## ✨ 核心能力
- 需求解析与架构设计：通过长链推理生成技术方案
- 自动化代码生成：快速实现核心功能，自动处理依赖与环境配置
- 智能调试：自主分析报错日志、复现问题并迭代修复
- 文档与部署：自动生成标准化 README 与部署脚本
